# Eurocamp Frontend - Vue.js Application

A modern, responsive Vue 3 frontend application for managing users, parcs, and bookings with the Eurocamp API.

## Prerequisites

- Eurocamp API running on `http://localhost:3001`

## Installation

```bash
# Navigate to the frontend directory
cd apps/frontend-vue

# Install dependencies
npm install

# Start development server
npm run dev
```

The application will be available at `http://localhost:3000`
